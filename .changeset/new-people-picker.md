Added new @equinor/fusion-wc-people package with comprehensive web component library for selecting and viewing people. Includes fwc-people-picker component for person selection with advanced search functionality, debounced search, keyboard navigation, and support for preselected people. Includes fwc-people-viewer component for displaying people in list or table format with responsive design. Features avatar customization with colors, service principal support, copyable names, and application badges.
