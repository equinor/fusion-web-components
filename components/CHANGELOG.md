### Major Changes

- [#2069](https://github.com/equinor/fusion-web-components/pull/2069) [`1a72d5a`](https://github.com/equinor/fusion-web-components/commit/1a72d5a1045610347cb55353a362d88167ecbfc8) Thanks [@eikeland](https://github.com/eikeland)! - Introduced new @equinor/fusion-web-components package - a unified collection package for Fusion web components. This package provides a convenient single entry point for importing Fusion web components, starting with the people components (PeoplePickerElement and PeopleViewerElement). The package re-exports components with proper TypeScript types, custom elements manifests, and subpath exports for flexible importing.
