# `@equinor/fusion-wc-page`

[![Published on npm](https://img.shields.io/npm/v/@equinor/fusion-wc-page.svg)](https://www.npmjs.com/package/@equinor/fusion-wc-page)

## Overview

`fwc-page` provides the foundation for composing application pages. It exposes dedicated regions for header, primary content, and footer content, allowing applications to build consistent page layouts without managing the overall page structure themselves.

## Installation

```bash
npm install @equinor/fusion-wc-page

## Description

Page web component for composing application pages with header, main content, and footer slots.
