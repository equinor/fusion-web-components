# Markdown Viewer

```sh
npm i @equinor/fusion-wc-markdown
```

## Example

```html
<fusion-wc-markdown-viewer>
  **some** markdown *text*
</fusion-wc-markdown-viewer>

<fusion-wc-markdown-viewer value="**some** markdown *text*">
</fusion-wc-markdown-viewer>
```

## Usage
```js
const markdown = "# my heading here";
<fwc-markdown-editor value={markdown} />
```

```html
<fusion-wc-markdown-viewer>
  **some** markdown *text*
</fusion-wc-markdown-viewer>
```
